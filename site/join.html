Join mycellm

Contribute your GPU to the distributed LLM inference network.

Network Invite

You've been invited to join a network.

1

Install

$ pip install mycellm

Requires Python 3.11+. Works on macOS, Linux, and Windows (WSL).

2

Initialize

$ mycellm init

Creates your identity and connects to the public bootstrap node. Auto-approved.

3

Start Serving

$ mycellm serve
# Or install as a service:
$ mycellm serve --install-service

Your node will appear on the network stats page.

D

Docker Alternative

docker run -d --name mycellm -p 8420:8420 -v mycellm-data:/data/mycellm ghcr.io/mycellm/mycellm

Auto-initializes on first run. Data persisted in the named volume.
